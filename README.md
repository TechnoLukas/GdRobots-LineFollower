## Features

- Random path generator (Using `Path3D`)

- Driving car with from turning wheels (Using `VehicleBody3D`)

- Grayscale sessonsors (Using `Camera` and `ImageTexture`)

- PID regulators (Using `Slider`)

- PID graph using `Line2D`
